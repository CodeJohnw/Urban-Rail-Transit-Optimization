# 🚇 Urban Rail Transit Operation Optimization

A complete solution for the 2023 MathorCup Mathematical Modeling Challenge Problem B.

This project demonstrates how passenger demand, train capacity, operational cost and service level can be integrated into a practical metro operation optimization model.

## Features

- OD matrix analysis
- Section passenger flow calculation
- Short-turn routing optimization
- Multi-objective optimization
- Pareto frontier analysis
- Automatic timetable generation
- Visualization dashboard

## Results

Optimal solution:

- Long route: Station1 → Station30
- Short route: Station5 → Station26
- Long route trains: 15
- Short route trains: 15
- Headway: 120 s
